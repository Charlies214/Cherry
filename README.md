# Cherry Studio 主题 CSS 收集仓库

## 📌 项目简介

本仓库用于收集和整理 Cherry Studio 的主题 CSS，方便用户快速获取和应用不同风格的界面主题。

⚠️ **注意：**

* 本仓库中的主题 CSS **均收集自网络**，主要来源于 **LinuxDo 论坛**
* **本人仅做收集整理，不是原创作者**
* 所有版权归原作者所有

---

## 📂 内容说明

仓库中包含：

* 各类 Cherry Studio 主题 CSS 文件
* 不同风格（简约 / 深色 / 护眼 / 个性化等）主题
* 持续收集与更新中

---

## 🚀 使用方法

1. 选择你喜欢的 CSS 文件
2. 复制其中的内容
3. 粘贴到 Cherry Studio 设置 -- 显示设置 -- 划到最下面自定义CSS 中
4. 保存并应用即可
